# ChineseCoupletsSmallProgram
微信小程序端
能通过简单的微信授权登陆
而通过手机账号注册、登陆的用户，能实现后期产品的登录信息统一，进行多端联动（项目长远规划）；

楹联相关功能
完整的搜索
用户分享
动态交流
